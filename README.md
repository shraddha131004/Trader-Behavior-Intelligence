# Trader Behavior & Market Sentiment Analysis

This project analyzes the relationship between trader performance and
market sentiment using historical crypto trading data and the Fear &
Greed Index.

## Objective

Explore how market sentiment impacts trader profitability and uncover
hidden behavioral patterns to drive smarter trading strategies.

## Datasets

-   historical_data.csv -- Trade-level data
-   fear_greed_index.csv -- Market sentiment index

## Methodology

-   Data cleaning & profiling
-   Sentiment regime classification
-   Performance correlation analysis
-   Behavioral pattern detection

## Key Insights

-   Sell trades outperform buy trades during fear regimes
-   Greed markets show higher volatility and lower consistency
-   Extreme sentiment regimes create momentum opportunities

## Conclusion

Sentiment-aware strategies outperform static trading models and provide
better risk control.
